## 5. 주요 기능 명세

### 5-1. 고정 내비게이션 바

- `position: fixed` 상단 고정
- 배경: `rgba(26,58,92,0.96)` + `backdrop-filter: blur(6px)`
- 모바일: 햄버거 버튼 → 드롭다운 메뉴
- 섹션 앵커 링크 (`#about`, `#career` 등)

### 5-2. 스크롤 페이드인 애니메이션

- `IntersectionObserver` API 사용
- `.fade` 클래스: `opacity: 0` + `translateY(28px)` → `.visible` 시 해제
- 임계값(threshold): `0.15`

### 5-3. 카운트업 애니메이션 (About 섹션)

- `data-target` 속성으로 목표값 지정
- 30프레임, 30ms 간격으로 증가
- 수치: `30+`년, `17+`편, `3+`권, `12+`개

### 5-4. 논문 아코디언

- 카테고리 4개: IT 거버넌스·전략 / AI·디지털 혁신 / IoT·서비스 / 사회·기술 정책
- 클릭 시 펼치기/접기, 동시에 하나만 열림

### 5-5. 문의 폼

- 수집 항목: 이름, 소속, 연락처, 문의 유형, 내용
- 문의 유형: `강의 의뢰` / `원고·저서 관련` / `심사위원 위촉` / `기타`
- 제출 시 `mailto:` 프로토콜로 메일 앱 연동 (서버리스)
- 개인정보 수집 고지 포함
